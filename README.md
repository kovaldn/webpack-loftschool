# Учебная сборка Loftschool

> Сборка работает на webpack 2.2.1

Используйте npm или yarn для установки зависимостей и запуска webpack

#### Для начала работы

1. ```clone this repo```
2. ```cd !$```
3. ```yarn install```  
4. ```yarn run start``` - for developing 
5. ```yarn run build``` - for production
